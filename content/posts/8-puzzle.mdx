# 8-puzzle solver

I recently built an interactive 8-puzzle game with three different solver strategies:

- **BFS** (Breadth-First Search)
- **Greedy Best-First Search**
- **A\* Search**

- Greedy approach takes way less execution time, but cannot find the optimal answer.
- A\* algorithm can effectively find good solutions with reasonable performance.
